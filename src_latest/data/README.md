# Data

This folder keeps only the small input datasets needed for the main simulation.

Included directories:

- `input/full_county_networks/`
- `input/subsample_networks/`
- `input/sampled_household_based/`

Excluded on purpose:

- raw reindex source CSV files
- `input/00_NYS_County_vax_rate_by_age.csv` (private)

Do not add the raw source files unless they are explicitly approved for upload.
